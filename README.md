
# TeamTrace

Full-stack SaaS for real-time workforce tracking and job-site monitoring.

## Problem
Companies struggle to track employee presence across multiple job sites.

## Solution
TeamTrace provides:
- GPS workforce tracking
- Job-site geofencing
- Shift monitoring
- Manager dashboards

## Tech Stack
Frontend: React
Backend: FastAPI
